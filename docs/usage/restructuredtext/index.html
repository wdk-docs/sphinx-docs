reStructuredText¶

reStructuredText(reST)是Docutils和Sphinx使用的默认纯文本标记语言。 Docutils提供了基本的reStructuredText语法，而Sphinx扩展了它以支持其他功能。

以下指南介绍了reST的最重要方面。有关权威的reStructuredText参考，请参阅 docutils documentation.
